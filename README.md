## HOW IT WORKS (FOR THE MORBIDLY CURIOUS)

When you invoke the shortcut, a content script is injected directly into the page's DOM — like a parasite burrowing into its host. It:

1. Summons a rename dialog from the void
2. Overwrites `document.title` with your chosen name
3. Deploys a `MutationObserver` — an ever-watchful sentinel that *destroys* any attempt by the page to reclaim its title

The page never stood a chance.
